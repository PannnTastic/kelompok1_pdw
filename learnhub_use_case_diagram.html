LearnHub E-Learning Platform - Use Case Diagram

Use Case Description

Student

Browse Courses: Students can explore all available courses on the platform.
View Course Details: Students can see detailed information about specific courses.
Sign Up: New users can create an account on the platform.
Login: Students can authenticate and access their account.
Enroll in Course: Students can register for a specific course.
Access Course Materials: Students can view learning content for enrolled courses.
Track Learning Progress: Students can monitor their advancement in courses.

Instructor

Create Course: Instructors can develop new courses on the platform.
Manage Course Content: Instructors can update course materials.
View Student Progress: Instructors can monitor student performance.
Provide Feedback: Instructors can give feedback to students.
Login: Instructors can authenticate and access their account.

Relationships

Include Relationships: Indicates that a use case includes the behavior of another use case. For example, "Enroll in Course" includes "Login" because a user must be logged in to enroll.
Extend Relationships: Indicates that a use case can be extended by another use case. For example, "View Course Details" can be extended by "Enroll in Course" when a student decides to enroll after viewing course details.
